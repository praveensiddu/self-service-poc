Allocated IPs

Allocated IPs displays the list of actual IP addresses that have been provisioned for L4 ingress for this purpose.

What you'll see

  • A comma-separated list of IPv4 addresses (e.g., 192.168.1.10, 192.168.1.11).
  • These are the specific IPs that your load balancers or ingress controllers can use.
  • The count should match the "Allocated" column.

Using these IPs

These IP addresses are configured in your cluster's L4 ingress resources and are used to route external traffic to your services. Reference these IPs when configuring DNS records or external load balancers.
