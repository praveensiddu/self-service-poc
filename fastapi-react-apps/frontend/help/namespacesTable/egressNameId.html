Egress Name ID

Egress Name ID is an optional identifier that links this namespace to an existing egress IP configuration. If set, outbound traffic for this namespace will be routed through the corresponding egress IP.

When to use

  • Use this when you need the namespaceâ€™s egress traffic to come from a specific IP.
  • Leave it empty if the namespace does not require dedicated egress routing.

Recommended default

By default, this field is prefilled with the application name to make it easy to align egress routing with the app.
