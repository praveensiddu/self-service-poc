Allocated

Allocated shows the number of L4 ingress IPs that have been successfully allocated for this purpose in the cluster.

Understanding this value

  • This represents the actual IPs that have been provisioned and are available for use.
  • When "Allocated" matches "Requested", the allocation is complete.
  • If "Allocated" is less than "Requested", you can use the "Allocate" action to provision the remaining IPs.

Actions

Use the "Allocate" button in the Actions column to automatically allocate additional IPs up to the requested amount.
