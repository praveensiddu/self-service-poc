Requested

Requested indicates the total number of L4 ingress IPs that have been requested for this purpose in the cluster.

Understanding this value

  • This represents the desired number of IPs needed for load balancing or high availability.
  • The value is configured in your infrastructure request files.
  • Compare this with the "Allocated" column to see if your request has been fulfilled.

Note

If the requested number is higher than allocated, you may need to allocate additional IPs using the allocation feature.
