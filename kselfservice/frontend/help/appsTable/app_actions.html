App Actions shows the operations available for each application in the Apps table. What you can do depends on your role and permissions.

Operation What it does App Manager Platform Admin
Edit App Update app metadata (for example, description). Yes Yes
ArgoCD View/add/update ArgoCD details for the application. Yes Yes
Request Access Request viewer/manager access to the application. Yes N/A (already has access)
Commit / Push and Open PR Push latest request state for this app/environment and create a PR to main. Yes Yes
Discard edits Discard local edits for this app/environment and restore the content from origin/main. If the app/environment does not exist in origin/main, it will be removed from your local view. Yes Yes
Delete App Delete the application entry for the selected environment. Yes Yes

Notes

  • Some actions appear disabled if you do not have manage permission for that application.
  • Platform admins typically have full access; the UI hides the Request Access button for platform admins.
